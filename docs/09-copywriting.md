# 09 Copywriting: Zeepro Website

## Purpose
To define the structural content requirements and store the finalized text for the Zeepro website. This document ensures the brand voice is consistent and that all text aligns with the B2B validation goals of the project.

## Scope
This document currently serves as a *Content Brief and Placeholder Matrix* for the v1 website. It outlines what text needs to be written for each page specified in `08-page-specifications.md`.

## Assumptions
- **Assumption 1: SEO-Informed, Not SEO-Driven.** We assume the copy must naturally include industry keywords (e.g., "UAE Construction," "Dubai Maintenance"), but it must prioritize human readability and B2B professionalism over aggressive keyword stuffing.
  - **Justification:** The primary goal is validation by human partners, not just ranking on Google. Keyword stuffing damages credibility.
- **Assumption 2: Client-Provided Core Logic.** We assume the client must provide the factual bullet points (history, services offered), which a professional copywriter will then polish.
  - **Justification:** The development and design teams cannot invent Zeepro's company history, technical capabilities, or safety records.
